    Sistema de Monitoramento de Dados Eólicos

    Bem-vindo à documentação do Sistema de Monitoramento de Dados Eólicos. Esta documentação fornece uma visão detalhada dos objetivos, funcionalidades e diagramas UML do sistema, facilitando o entendimento e a utilização da plataforma por parte de desenvolvedores, administradores e usuários finais.

    Estrutura da Documentação

    A documentação está organizada da seguinte forma:

    1. Objetivos do Sistema: Descreve os objetivos primários, secundários e terciários que a plataforma pretende alcançar, garantindo uma visão clara dos propósitos e expectativas do sistema.

    2. Funcionalidades do Sistema: Detalha as funcionalidades principais disponibilizadas tanto para usuários quanto para administradores, explicando como cada funcionalidade contribui para o cumprimento dos objetivos do sistema.

    3. Diagramas UML: Apresenta uma série de diagramas UML (Diagrama de Caso de Uso, Diagrama de Classes, Diagrama de Sequência e Diagrama de Atividades) que ajudam a visualizar a estrutura e o fluxo de dados dentro do sistema.

    Conceitos Iniciais

    Plataforma de Monitoramento

    A plataforma de monitoramento de dados eólicos é um sistema desenvolvido para auxiliar na gestão e análise de dados provenientes de parques eólicos. Este sistema permite o envio de arquivos CSV contendo dados de medição, a formatação e interpretação desses dados, e a geração de gráficos e relatórios para análise detalhada.

    Usuários e Administradores

    • Usuários: Enviam arquivos de dados, visualizam gráficos gerados e baixam arquivos formatados para envio a entidades reguladoras.
    • Administradores: Cadastram novos usuários, fornecem treinamento e monitoram os dados enviados de forma centralizada.

    Sensores Utilizados

    Os principais sensores utilizados para a coleta de dados eólicos incluem: - Windvane (biruta) - Anemômetro - Termohigrômetro - Barômetro - Datalogger Ammonit

    Notificações e Alertas

    O sistema possui um mecanismo de notificações que alerta os usuários sobre a necessidade de envio de dados e notifica os administradores em caso de dados fora do padrão esperado. Este mecanismo é fundamental para manter a integridade e a consistência dos dados coletados.

    Como Navegar na Documentação

    Utilize o menu de navegação à esquerda para acessar as diferentes seções da documentação. Cada seção contém informações detalhadas que ajudarão no entendimento e utilização eficaz do sistema.
